## Props

> ⚠️ **Stability: Beta.** The Checkbox Figma info card carries a **Beta** badge. Expect API churn.

### `Checkbox` (default export)

| Prop | Type | Default | Required | Description |
|------|------|---------|:--------:|-------------|
| `testId` | `string` | `'CHECKBOX'` | | Checkbox test id |
| `id` | `string` | autogenerated | | Value for `id` input attribute and `htmlFor` label attribute |
| `label` | `node` | `null` | | Label content for the checkbox input (text or custom component) |
| `value` | `number \| string \| bool \| shape({ id*: string })` | `''` | | Checkbox input value |
| `name` | `string` | — | | Name prop for the checkbox input |
| `infoBoxText` | `node` | — | | Shows info icon and renders text inside a Tooltip |
| `infoBoxButtonLabel` | `string` | — | | Accessible aria-label for the info button |
| `isChecked` | `bool` | `false` | | Whether the checkbox is checked |
| `isDisabled` | `bool` | `false` | | Whether the checkbox is disabled |
| `isIndeterminate` | `bool` | `false` | | Whether the checkbox is in the indeterminate state |
| `showLabelTooltipOnOverflow` | `bool` | `true` | | Show label tooltip when the label overflows |
| `onBlur` | `func` | `noop` | | Synthetic blur event |
| `onChange` | `func` | `noop` | | Callback for the checkbox input |
| `onFocus` | `func` | `noop` | | Synthetic focus event |

### `CheckboxGroup` (named export)

Part of the same `@8x8/oxygen-checkbox` package. Wraps multiple `Checkbox` components.

| Prop | Type | Default | Required | Description |
|------|------|---------|:--------:|-------------|
| `isHorizontal` | `boolean` | — | | Lay the group out horizontally instead of vertically |

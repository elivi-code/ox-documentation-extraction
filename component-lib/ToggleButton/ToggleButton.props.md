## Notes

- `ToggleButton` is a **controlled component** — manage `isChecked` in the parent and update it via `onChange`.
- `id` is autogenerated if not supplied; supply it explicitly when you need a stable `htmlFor` / `aria-labelledby` pairing.
- `infoBoxText` + `infoBoxButtonLabel` must be supplied together — the button label is required for accessibility when the info icon is shown.
- `value` accepts `number | string | bool | { id: string }`. The object form `{ id: string }` is for **group-item identification**: pass `value={{ id: 'my-id' }}` when the `onChange` handler needs to map a toggle back to a record by stable ID (e.g. rendering toggles from a data array and looking up the row on change).
